Ship prompts like you ship code.

Version, test, and score system prompts in the browser. No code changes needed.

The eval studio for AI agents. Decorate your agent function, mount the UI, and iterate on prompts with real test cases and AI judges.

Get Started Try Live Demo GitHub
app.py 
from biscotti import Biscotti, biscotti

@biscotti(name="my agent", default_system_prompt="You are helpful.")
async def my_agent(user_message: str, system_prompt: str) -> str:
    # Your existing agent code
    ...

bi = Biscotti()
app.mount("/biscotti", bi.app)

Version Every Change

Save iterations with notes, compare side-by-side, and roll back when needed.

Run Real Test Cases

Define test inputs, run them against your live agent, and see the impact.

Score with AI Judges

Auto-generate criteria, batch-evaluate, and track scores across versions.

Sweep with Bulk Runs

Run a matrix of models, temperatures, and test cases in one click. Stream results, auto-score, export CSV.

Improve with Coach

After an eval, get AI-suggested prompt edits targeted at the criteria that scored lowest.

For Developers

Add to your project in 60 seconds

1 Install

terminal 
uv add biscotti

2 Decorate your agent

app.py 
from biscotti import Biscotti, biscotti

@biscotti(name="my agent", default_system_prompt="You are helpful.")
async def my_agent(user_message: str, system_prompt: str) -> str:
    # Your existing agent code
    ...

bi = Biscotti()
app.mount("/biscotti", bi.app)

3 Open the UI

terminal 
biscotti dev
