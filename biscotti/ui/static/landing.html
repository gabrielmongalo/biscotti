biscotti.

The prompt eval studio for AI agents

Iterate on system prompts with real test cases, track every version,
and score quality with AI judges. Built for prompt engineers, not just developers.

Open Studio GitHub

Features

Ship better prompts, faster

Version Every Change

Save each prompt iteration with notes. Compare any two versions side-by-side and roll back to what worked.

Run Real Test Cases

Define test inputs, run them against your live agent, and see exactly how different prompts and models affect the output.

Score with AI Judges

Auto-generate scoring criteria from your prompt and batch-evaluate every test case. Track quality scores across versions to see what improved.

Improve with AI Suggestions

Generate eval criteria and get AI-driven feedback on your prompts. Understand why scores are low and what to change.

For Developers

Add to your project in 60 seconds

Install

terminal 
uv add biscotti
Works with Claude Code -- Claude can help you integrate, write tests, and debug
terminal 
biscotti init-claude

Add to your project

app.py 
from biscotti import Biscotti, biscotti

@biscotti(name="my agent", default_system_prompt="You are helpful.")
async def my_agent(user_message: str, system_prompt: str) -> str:
    # Your existing agent code
    ...

bi = Biscotti()
app.mount("/biscotti", bi.app)
